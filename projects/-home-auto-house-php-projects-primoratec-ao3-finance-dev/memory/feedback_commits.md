Nunca fazer `git commit`, `git push` ou qualquer operação de escrita no repositório sem antes pedir confirmação explícita do usuário.

**Why:** Usuário quer ter controle total sobre o que é commitado e quando.

**How to apply:** Sempre que estiver prestes a criar um commit ou fazer push, parar e perguntar ao usuário antes de executar qualquer comando git destrutivo ou de escrita.
